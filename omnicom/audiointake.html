Mix Review

There are two review approaches:

Unsupervised Mix Review - The engineer delivers files for review. Creative/client provides feedback asynchronously (e.g., via Frame.io).

Supervised Mix Review - A live review session where creative/client attends via video stream and provides real-time feedback to the mixer (e.g., Evercast). This does not replace the initial mix pass, but changes how review is conducted.

Delivery

This is not the deadline for receiving a mix to review. This represents the final ship date of the project.
