Mix Reviews

There are two review approaches:

Unsupervised Mix Reviews - The engineer delivers files for review. Creative/client provides feedback asynchronously via Frame.io.

Supervised Mix Reviews - A live review session where creative/client attends via video stream and provides real-time feedback to the mixer via Evercast. This does not replace the initial mix pass, but changes how review is conducted.
